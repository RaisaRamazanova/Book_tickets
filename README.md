# Book tickets

This is an e-commerce app for book tickets using flutter for iOS and Android

**Project uses**: 
  - Flutter 3.10
  - Third party libraries
  - Getx Package (Getx package to manage the state and routing of the app. Also used Getx for api client)
  - Working with assets
  - List View Builder
